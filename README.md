## AENIGMA MARK 2.3
---
About

AENIGMA is a riddle game.

There are 23 levels.

Each level has one correct solution.

The first levels are straightforward.

The later ones are not.

Do not assume that every puzzle follows the same logic, or that the same approach will work twice.

Sometimes the answer is hidden in plain sight.

Sometimes it is hidden in the question itself.

If something seems impossible, you are probably looking in the wrong place.
---
Rules

- Every level has a single intended solution.
- There are no time limits.
- Nothing needs to be installed.
- Think before you type.
- If you're stuck, keep looking.
  --
Goal

Reach Level 23.

Good luck.

You'll need it.
